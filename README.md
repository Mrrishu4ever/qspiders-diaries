# The QSpiders Diaries 📖🚆

> Salem, Tamil Nadu se Bangalore tak ka safar — QSpiders Hebbal ki class, dost, mentors, aur wo yaadein jo hamesha ke liye reh gayin.

A cinematic, scroll-driven storytelling webpage built with React (no build step — runs straight in the browser via CDN). Chapter-by-chapter journey, complete with a travelling train motif, curtain-reveal transitions, custom cursor, ambient particles, and original shayari cards.

## Live Website

Read the story here: https://mrrishu4ever.github.io/qspiders-diaries/

## Tech Stack

- React 18 (via CDN, no bundler needed)
- Babel Standalone (in-browser JSX)
- Vanilla CSS (custom cursor, curtain transitions, 3D tilt, particle system)
- Google Fonts — Fraunces, Work Sans, Caveat

## Project Structure

- index.html — the full story, open directly in any browser
- qspiders_diaries.pdf — original written diary (source material)
- README.md — this file

## Run It Locally

    xdg-open index.html

## What's Inside

- 6 chapters, each with its own curtain-reveal entrance animation
- A "train" that travels down the page as you scroll — the story's own Salem to Bangalore journey
- Character cards for every trainer and friend in the story (flip to see their stats)
- Original shayari at the end of every chapter, downloadable as shareable cards
- Night Diary mode, ambient dust particles, custom cursor, and more
